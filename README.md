# Phase2_ Brand Guidelines

A brand guidelines site and UI component library built with Next.js, shadcn/ui, and Tailwind CSS. Deployed to GitHub Pages.

## Prerequisites

Requires **Node.js v20 or higher**. If you use nvm:

```bash
nvm use 20
```

If you don't have Node 20 installed:

```bash
nvm install 20 && nvm use 20
```

## Getting Started

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Deploy to GitHub Pages

```bash
npm run build
```

The static export will be in the `out/` directory. GitHub Actions handles deployment automatically on push to `main`.
